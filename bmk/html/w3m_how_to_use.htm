w3m使用方法简介

q/Q

退出，Q不需要确认

移动

 

SPACE

向前翻页

b

向后翻页

h

左移光标

l

右移光标

j

下移光标

k

上移光标

J

向下翻滚一行

K

向上翻滚一行

z

将当前行置于屏幕中间

(

取消光标的移动（undo）

)

恢复之前的移动（redo）

链接

 

]

跳到最后一个链接

[

跳到第一个链接

TAB

跳到下一个链接

u

显示链接的url

ENTER

跳转到链接

B

回到之前的页面

标签操作

 

Ctrl-t

在新标签页打开链接

Ctrl-q

关闭当前标签

}

跳到下一个标签

{

调到前一个标签

其他操作

 

v

查看html源码

Ctrl-c

取消当前操作

a

光标在链接处时，保存链接到文件

光标在输入框时，可输入文本

=

显示当前文档的信息（在新开的页面上显示，信息很详细）

c

仅显示当前文档的地址

E

编辑本地源文件

m

是否支持鼠标（可使用鼠标滚轮上下翻页）
