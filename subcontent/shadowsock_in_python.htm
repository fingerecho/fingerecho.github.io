Shad0w-s0ck 在python下的使用

具体看python的第三方库 在 https://pypi.org/project/shadowsocks/

之前看见其他的朋友配置shad0ws0ck用了一大堆的shell指令，真替他遗憾，python明明有这样的第三方库，直接一键搞定(两个命令就搞定了)，非要整这样那样的配置，服务器不稳定换了还得文件拷贝来拷贝去，内容改来改去，真是既麻烦又浪费时间，写这篇文章主要是最近目标比较明确了，思路更清晰了些，所以有时间折腾一些看似不太重要的，看似容易实现的，以后也会经常有这样的

也是为了这篇文章更充实些,复制粘贴了一下

Usage:
ssserver -p 443 -k password -m rc4-md5
To run in the background:
sudo ssserver -p 443 -k password -m rc4-md5 --user nobody -d start
To stop:
sudo ssserver -d stop
To check the log:
sudo less /var/log/shadowsocks.log
