关于tengine的一些使用心得

据说 tengine的性能是原版nginx的1.5倍 或者说tengine提高了nginx的30%的性能，趁着最近稍微有点时间，把nginx上所有跑的配置全部迁移到了tengine上，效果相当不错(自我感觉)，兼容的也相当好。 原先跑的uwsig_proxy 的东西在tengine上没有任何问题。 所以干脆nginx -s stop

官方地址：http://tengine.taobao.org/documentation.html

按照一般的操作 configure --prefix=/the path you want to install/ ...

make && make install

另外根据自己的nginx 使用习惯 ，/tengine目录下也一样是有一个 nginx.conf 与nginx的是一样的 在 /config 下创建一个 conf.d 然后在 nginx.conf 中 include 即可。
